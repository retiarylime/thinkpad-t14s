# auto-cpufreq

Source: [https://github.com/AdnanHodzic/auto-cpufreq](https://github.com/AdnanHodzic/auto-cpufreq)

- Automatic CPU speed & power optimizer for Linux.
- For my machine, `auto-cpufreq` provides better battery life than `tlp`.

## Installation

1. Install [auto-cpufreq](https://github.com/AdnanHodzic/auto-cpufreq?tab=readme-ov-file#installing-auto-cpufreq)
2. Copy [auto-cpufreq.conf](auto-cpufreq.conf) into `/etc/auto-cpufreq.conf`

```bash
sudo cp auto-cpufreq.conf /etc/auto-cpufreq.conf
```

3. `auto-cpufreq` GUI will automatically installed. You can view and edit the governor and CPU turbo settings.
